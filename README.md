# Chocadeira caseira feita com Arduino utilizando controlador PID

Chocadeira Caseira com controle automático de temperatura
O que foi utilizado até agora:
Arduino Uno
Dimmer 
Lâmpada 60w
Sensor bmp180
2 coolers 5V
